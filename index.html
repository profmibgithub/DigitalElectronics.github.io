Course-Level Learning Outcomes

By the end of the course, students should be able to:
  • Convert between decimal, binary, octal, hexadecimal; perform signed/unsigned arithmetic and code representations (BCD, parity)
  • Analyze and synthesize combinational logic using Boolean algebra, DeMorgan’s theorems, SOP/POS, and Karnaugh maps (≤4 variables)
  • Design, simulate, and test combinational blocks: adders, comparators, decoders/encoders, multiplexers/demultiplexers
  • Explain and implement latches, flip-flops, registers, counters, and simple timing with 555 timer
  • Describe CPU organization (von Neumann), buses, memory, I/O (polling, interrupts, DMA), and instruction types; contrast microprocessors, microcontrollers, and SoCs
  • Implement specified modules in Verilog/SystemVerilog with testbenches, including a simple 4-bit ALU

    Course Syllabus

    Complete curriculum covering fundamental to advanced concepts

    Theory Units

    • Digital/Analog Quantities, Binary Digits, Logic Levels, Waveforms, Clock/Timing Diagrams
    • Number Systems (Decimal, Binary, Octal, Hexadecimal) & Conversions
    • Unsigned/Signed Arithmetic, Complements, BCD, Parity Code
    • Logic Gates (AND, OR, NOT, NAND, NOR, XOR, XNOR)
    • Boolean Algebra, DeMorgan's Theorems, Simplification

    • SOP/POS forms, K-Maps (up to 4 variables), Don't care conditions
    • Combinational Logic Circuits (AND-OR, AND-OR-INVERT, etc.), Universal Gates
    • Half/Full Adder, 4-bit Adder, Comparator, Decoder, Encoder, Multiplexer, Demultiplexer

    • Latches (SR, D, Gated)
    • Flip Flops (vs. Latch, Level/Edge-Triggered, D, JK)
    • Flip Flop Applications (storage, counting), 555 Timer
    • Shift Registers (Serial/Parallel 4-bit)
    • Counters (Synchronous/Asynchronous 2/3 bit), Decade/Johnson Counter

    • von Neumann Architecture, CPU, Memory, I/O, Buses, Bus Master/Contention, System Timing
    • Microprocessor, ALU, Control/Timing Unit, Registers, Instruction Cycle
    • Memory operations, I/O (Polling, Interrupts, DMA), CPU Instruction Types
    • Microcontrollers, SoC

    Practical Experiments (Verilog/SystemVerilog)

    Basic Gates Implementation

    XOR Gate Design

    Design and simulate a 3-to-8 Decoder

    8
    D Flip Flop

    9
    4-bit ALU Design

    Design and simulate a simple 4-bit ALU (ADD, SUB, AND, OR)

    Course Assignments

    Four comprehensive assignments to evaluate your understanding

    Assignment 1: Presentation

    Research & Present
    Emerging Technologies in Digital Electronics

    Students will research and present on topics like quantum computing, neuromorphic chips, or advanced VLSI technologies

    Duration: 15-20 minutes
    Deliverables: PowerPoint/PDF, Research report, Live presentation
    Due: TBA

    Assignment 2: Quiz

    Online Assessment
    Digital Logic Fundamentals Quiz

    Online quiz covering Units 1-2 concepts including number systems, logic gates, and combinational circuits

    Duration: 60 minutes
    Format: Multiple choice and short answer questions
    Due: TBA

    Assignment 3: Debate

    Team Discussion
    Future of Digital Electronics: Hardware vs Software Solutions

    Teams debate the role of dedicated hardware vs software solutions in emerging technologies

    Format: Team-based debate
    Duration: 30 minutes per team
    Due: TBA

    Assignment 4: Practical

    Design Project
    Verilog Design Project

    Design and implement a complete digital system using Verilog (e.g., simple calculator, traffic light controller)

    Language: Verilog/SystemVerilog
    Deliverables: Code, Simulation results, Documentation, Demo
    Due: TBA
